# 💬 Projeto de Chat ao Vivo — Site e App

Projeto desenvolvido durante curso de Python com foco em desenvolvimento web e mobile.

## 📌 Sobre o Projeto

Desenvolvimento de um sistema de chat ao vivo para site e aplicativo, com front-end e back-end construídos inteiramente em Python usando a ferramenta Flet.

## ✅ Resultado

- Chat funcional em tempo real para web e mobile
- Interface desenvolvida com Flet (front-end em Python)
- Back-end integrado na mesma aplicação

## 🛠️ Tecnologias Utilizadas

- Python
- Flet

## ▶️ Como Executar

```bash
# Clone o repositório
git clone https://github.com/taliaalmeida/projeto-site-app

# Instale as dependências
pip install flet

# Execute a aplicação
python main.py
